# Percento

![Percento social preview](./public/site.png)

Percento is a fast, minimal percentage calculator app with six tools side-by-side for instant answers.

## Features

- Percentage of a number (X% of Y)
- What percentage (X is what % of Y)
- Percentage change (from X to Y)
- Find total (X is Y% of what)
- Percentage difference (between A and B)
- Increase/decrease value (X ± Y%)

## Run locally

Prerequisites:

- Node.js (v18+ recommended)
- npm

Setup:

```sh
npm install
npm run dev
```

Then open http://localhost:4321

## Useful scripts

```sh
npm run build
npm run preview
```
